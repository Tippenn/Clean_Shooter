# 🎮 Shooter AI Demo (Unreal Engine)

This project is a third-person / FPS-style shooter built in Unreal Engine, focusing on **AI behavior systems** and **animation integration** using C++ and Blueprint.

---

## 📌 Overview

In this project, I implemented core gameplay systems including:
- Player shooting mechanics
- Enemy AI with decision-making behavior
- Animation systems using Blend Spaces
- GameMode extensions

The main goal of this project is to demonstrate my understanding of **Unreal Engine gameplay architecture**, especially AI systems and animation workflows.

---

## 🧠 Key Features

### 🔫 Combat System
- Player-controlled shooting mechanics
- Basic weapon handling and hit detection

### 🤖 AI System (Behavior Tree)
- Implemented using Behavior Tree & Blackboard
- Custom:
  - Tasks
  - Services
  - Decorators
- AI can:
  - Detect player
  - Move strategically
  - Engage in combat

### 🎭 Animation System
- Blend Spaces for smooth movement transitions
- Animation Blueprint integration
- State-based animation handling

### 🧩 Game Framework
- Extended GameMode class
- Organized gameplay logic using C++
